# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please report it. Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested mitigation (if any)

We will acknowledge receipt within 48 hours and work with you to resolve the issue promptly.

## Responsible Disclosure
We appreciate responsible disclosure. Do not publicly disclose the vulnerability until it has been fixed and we have given permission to do so.
